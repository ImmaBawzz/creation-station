# Creation Station v1.1 - Release Hardening

## Goal
Make the existing v1.0 workflow more stable, clearer, and easier to use without expanding product scope.

## Non-goals
- No v2.0 features
- No external integrations
- No cloud sync
- No major architecture rewrite
- No new AI agents unless required for stability

## Priority Fixes
1. Move / redesign the Idea Inbox Apply button.
2. Improve To Do Task Board organization.
3. Add task grouping, labels, collapsible layers, or filters.
4. Verify no regressions in Idea -> Plan -> Review -> Approval -> Task flow.
5. Add or update QA checklist.

## Definition of Done
- Core workflow works from empty state to approved task.
- Task board remains readable with many tasks.
- No accidental deletion or branch confusion.
- UI feels stable enough for repeated daily use.
