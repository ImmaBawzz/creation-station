# Do Not Add Yet

This file exists to protect the project from expansion while tired or distracted.

## Not Now

- Agent command center
- Agent meetings
- Calendar scheduling
- ComfyUI connector
- GitHub connector
- VSCode automation connector
- Asset vault
- Plugin marketplace
- Team system
- Cloud sync
- Payment system
- User accounts
- Deployment pipeline

## Why

The core loop must become boringly reliable first:

Idea → AI Factory Plan → Review → Revision → Approval → Tasks

Expansion before stability will create a beautiful mess that is hard to trust.

## Unlock Condition

Only begin v0.6 after:

- [ ] v0.5.1 passes QA
- [ ] Core loop works 3 times in a row
- [ ] Code is committed
- [ ] Agent run report is reviewed
- [ ] Human approval is given
