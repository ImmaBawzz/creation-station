# Creation Station v0.5 Scope Lock

## Current Mission

Stabilize the core workflow:

Idea → AI Factory Plan → Review → Revision → Approval → Tasks

## Allowed Work

- Fix broken imports
- Fix Prisma/database issues
- Fix AI/Ollama connection issues
- Improve Review Inbox clarity
- Improve Task Board clarity
- Add small UI labels
- Add empty states
- Add error messages
- Make the full loop reliable

## Not Allowed Yet

- Asset vault
- Agent meetings
- Plugin system
- External connectors
- ComfyUI integration
- VSCode automation
- Calendar integration
- Team accounts
- Marketplace
- Multi-agent orchestration
- New major database models

## Definition of Done

The app is stable when:

1. User can create an idea.
2. User can send idea to AI Factory.
3. AI plan is saved.
4. Plan appears in Review Inbox.
5. User can request revision with notes.
6. User can re-plan with revision context.
7. User can approve plan.
8. Tasks are generated from nextActions.
9. Task Board displays generated tasks.
10. No major errors appear in terminal or browser.
